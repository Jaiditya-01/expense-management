# Expense Management Frontend

React/Vite frontend for the Expense Management System.

For full project setup, environment variables, sample accounts, and backend instructions, see the root [README.md](../README.md).

## Local Commands

```powershell
npm.cmd install
npm.cmd run dev
npm.cmd run lint
npm.cmd run build
```

Default local URL:

```text
http://localhost:5173
```

Required `.env`:

```env
VITE_API_URL=http://localhost:5000/api
```
